Privacy Policy

Effective date: 2025-05-01

1. Introduction

Load Time Tracker ("the Extension") is a browser add-on designed to help developers and QA teams measure and record how long selected websites take to load. This policy explains what data is collected, how it is used, and your rights.

2. Data Collected

The Extension only collects, processes, and stores the following data on your local machine :

  • Domain names you explicitly add to track (e.g. example.com).
  • Load timestamps: the exact moment a tracked page finishes loading.
  • Load durations: the time (in milliseconds) between navigation start and the page's load event.
  • Favicon URLs scraped from <link rel="icon"> tags for display only.
  • In-flight timing data: transient start-timestamps used to display live counters while a page is loading.

3. Purpose of Collection

All data is used solely to:

  • Display a live timer during page loads for domains you've chosen.
  • Aggregate total load time over sliding windows (last hour, day, week, and month).
  • Show site-specific favicons in the popup UI.
  • Automatically prune records older than 30 days to keep data fresh.

4. Data Storage & Retention

Data is stored in your browser's chrome.storage.local area. No data leaves your computer. Records older than 30 days are automatically deleted by the Extension.

5. No External Sharing

The Extension does not send any data—neither raw nor aggregated—to any server, analytics provider, or third party. There is no external "backend."

6. Security

All code runs locally under your control. We follow these principles:

  • No remote code execution: all scripts are packaged with the Extension. (See "Remote Code" below.)
  • Minimal permissions: we only request the bare minimum Chrome APIs needed for functionality.
  • Data isolation: all data remains in your Chrome profile's storage.

7. Your Choices & Controls

  • Add/remove domains: Use the "＋" and "×" buttons in the popup to manage which sites are tracked.
  • Clear all data: Removing a domain deletes all its logs and favicon entries. Uninstalling the Extension clears all storage.
  • Review storage: You can inspect chrome.storage.local via Chrome's DevTools under "Application → Storage → Extensions."

8. Changes to This Policy

We may update this policy to reflect new features or legal requirements. We will update the "Effective date" at the top accordingly. Your continued use after changes implies acceptance.
